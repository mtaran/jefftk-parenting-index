# Jeff Kaufman on Parenting — an unofficial index

A small, filterable index of [Jeff Kaufman's parenting notes](https://www.jefftk.com/news/kids).

**⚠️ These essays are not mine.** Every post is written by **Jeff Kaufman** and lives
on [his own site](https://www.jefftk.com/news/kids). This repo is only an unofficial,
fan-made index to help find the good ones. The summaries and topic/age tags are a
reader's condensed notes; each entry links back to Jeff's original, which is where the
real writing has always lived and will stay. Not affiliated with or endorsed by the author.

## What it is

A single self-contained `index.html` page (no build, no dependencies) that lets you:

- **Search** across all 213 posts by title and summary
- **Filter** by topic (independence, safety, sleep, behavior, food, etc.), by kid age
  (baby → tween), and by type (essay, recipe, DIY, review…)
- **Sort** best-first using a subjective 1–5 "gem" rating, or show **gems only (★4+)**
  to skip the recipes, photo dumps, and logs

Each card links straight to the original post on jefftk.com.
